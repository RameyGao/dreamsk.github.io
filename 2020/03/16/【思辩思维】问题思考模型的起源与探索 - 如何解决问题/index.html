【思辩思维】问题思考模型的起源与探索 - 如何解决问题

发表于 2020-03-16 | 分类于 思辩思维

在写这一篇之前，我问了自己一个问题，“在面对如何解决问题之前，我们是否应该回答另一个问题，‘问题是否应该被解决’”。面对这样的问题，很快便得到了答案。

  • 问题的定义就已经确定了，人有问题的关系是密切不可分的，即，只要人存在，拥有意识，就能产生问题。

  • 由问题的由来也可以得到问题，基于期望和现状之间的关系而产生的事物，即是主观的意识到客观的现实之间的联系。

  • 第三、即便我们认为问题存在，而不去解决也是无意义的。因为去回忆问题的同时，思维已经在将期望和现状进行调整，使其平衡。

所以以上任意一点都可以得出，即便人不去解决问题，也是会被问题关联着，由人的意识和现实影响，如果不解决，会使得人越来越不平衡，从而产生各种挫败感，类似于自卑，虚荣都是由问题挤压过多而产生，所以理应去解决，这其中不存在主动和被动的关系。所以就省去了“问题是否应该被解决”这样的讨论。

我们如何去解决问题呢？相信我们的前辈肯定做了大量的研究，使得我们现在拥有较多的经验和积累，去应对各种各样的问题。但由于问题的特殊性，以及人与人之间的复杂性，同一个解决办法不一定适合每个人。但这些经验和积累能够留存下来，势必在人的期望和现状中找到了一定的平衡点，并且人也愿意为之牺牲自己的一小部分期望，降低自己对问题被解决的要求。而此时的降低要求，也就会对未来的某些问题留下隐患，更严重的会对自己的性格造成改变。

这样的说法是否在否定前辈们的研究呢？当然是，不。在一定程度上，我们并没有对前辈们留下来的经验和积累进行深度思考和分析，而只是在模仿和套用前辈们的做法。有时候，看似解决了问题，却带来了一些的隐患（此处并不是不去使用前辈们的方法，互相借鉴是有帮助的，但需要抓住核心）。

那么，从我对问题分析的角度，所建立的问题解决办法是FFBAF模型，一个基于人的期望和现状之间缩小差异，直至产生平衡的问题思考模型。

接下去将会具体从五个方面论述解决问题的具体步骤和方法。

  1. Face with the problem
  2. Find other problems
  3. Borrow some problems
  4. Ask some problems
  5. Forget the problem
