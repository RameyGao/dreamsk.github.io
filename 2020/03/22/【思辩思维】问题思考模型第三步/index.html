【思辩思维】问题思考模型第三步

发表于 2020-03-22 | 分类于 思辩思维

第三步：借用问题，B（Borrow some problems）

如果将问题局限在某一个固定的思维上的话，那么很可能会陷入循环。碰到这样一个情况的话，其实大部分人是利用线性思维在思考问题，这样的思维会有致命的缺陷，因为它是问题到答案，是点对点的，很容易出现绕进胡同出不来的状态。而此时我们需要跳出这个怪圈，去寻找其他解决办法，而这办法就是借用问题。大部分情况会消耗一些时间，但花费这些时间是有意义的。

首先，我们需要分析当前问题，并借用与之相关的问题。对于解决当前问题，表面上可能会觉得是没有多少作用，但是却非常关键，因为他会为解决当前问题提供很好的借鉴意义。而问题的种类有限（分类例如：工作问题、学习问题、健康问题…），也能够为借用相关问题提供较为明确的选择。

其次，尝试解决与之相关的问题，找联系。同一种类的问题并不是都不能解决，在解决的过程中，也一定留下了宝贵的经验或教训，那这些也便能够为我们解决当前问题提供良好的支撑。我们只需要从同类问题中找联系。如果同类问题没有解决，尝试去解决，因为它能产生灵感，起着重要作用，但别忘了我们的重点并不是为了解决同类问题，而是从同类问题中找联系，寻求解决方法，所以不需要执泥于此。

最后，依靠找到的联系，再去解决当前问题。这步能够带来更多的是回顾、反思，提供多角度的同类问题的分析过程和答案，再去解决当前问题会容易得多（一般解决心理问题会更加有效果）。
